Toggle-Switch

Switches toggle the state of a single setting on or off. They are the preferred way to adjust settings on mobile.

                        .bui-switch-toggle {
                            position: relative;
                            display: inline-block;
                            width: 30px;
                            height: 14px;
                            top: 3px;
                        }
                
                        .bui-switch-checkbox {
                            opacity: 0;
                        }
                
                        .bui-switch-slider {
                            position: absolute;
                            cursor: pointer;
                            top: 0;
                            left: 0;
                            right: 0;
                            bottom: 0;
                            background-color: #CAC9C7;
                            transition: .4s;
                        }
                
                        .bui-switch-slider:before {
                            position: absolute;
                            content: "";
                            height: 12px;
                            width: 12px;
                            left: 1px;
                            bottom: 1px;
                            background-color: #FFFFFF;
                            transition: .4s;
                        }
                
                        .bui-switch-checkbox:checked+.bui-switch-slider {
                            background-color: #00568F;
                        }
                
                        .bui-switch-checkbox:checked+.bui-switch-slider:before {
                            transform: translateX(16px);
                        }
                
                        .bui-switch-slider.bui-switch-round {
                            border-radius: 8px;
                        }
                
                        .bui-switch-slider.bui-switch-round:before {
                            border-radius: 50%;
                        }
                
                        .bui-switch-label {
                            padding-right: 10px;
                            color: #55565A;
                            font-size: 14px;
                        }
