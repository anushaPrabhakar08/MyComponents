Tooltip

        Tooltips display informative text when users hover over, focus on, or tap an element.
        When activated, tooltips display a text label identifying an element, such as a description of its function.

Upload1

      
.bui-tooltip .bui-tooltiptext
  {
    visibility: hidden;
    width: 104px;
    background-color: #000000;
    color: #fff;
    font-family: Helvetica Neue Regular;
    font-size: 14px;
    text-align: center;
    border-radius: 4px;
    /* padding: 8px 0; */
    margin: 8px 0;
    padding-left: 8px;
    padding-right: 8px;
    height: 24px;
    padding-top: 8px;
    /* Position the tooltip */
    position: absolute;
    opacity: 0.5;
  }
  
  .bui-tooltip:hover .bui-tooltiptext
    {
      visibility: visible;
    }

  .bui-tooltip-image,.bui-tooltip-upload 
    {
      display: flex;
      flex-direction: column;
      text-align: center;
      margin-bottom: 18px;
      margin-left: 45px;
      cursor: pointer;
    }
